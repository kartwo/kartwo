## 2. 里程碑路线图（详细 DoD 与验收）

**所有里程碑合主干前都过 CI 安全门禁（依赖+密钥扫描无高危）。**

### M0 · 地基与骨架
Repo `git init`、Go module、目录结构、配置加载、**数据层抽象选型落地**、纯 SQL 迁移框架、日志、基础 HTTP server(安全响应头)、embed Admin SPA 占位、健康检查、CI(govulncheck+密钥扫描)、生成 `CONVENTIONS.md`/`DECISIONS.md`/`LICENSES.md`/`PROGRESS.md`、lint。
- 前置决策：数据层 sqlc/ent/bun。
- DoD：`go build` 出单二进制；健康端点可访问；迁移建表；CI 绿。
- 验收：下载启动 → `/health` 正常 → 空库已建表。

### M1 · 核心数据模型 + Admin 基础 + 媒体上传
商品/**变体**/分类/库存/订单/客户 schema + 迁移；Admin API CRUD；Admin SPA 登录 + 商品管理；向导骨架(建管理员+设主口令)；**媒体上传**（本地存储+压缩/WebP/去 EXIF/内容哈希+上传安全）；**StoragePolicy 接口+默认不限额+磁盘监控/优雅降级+单文件护栏+孤儿清理**。
- 前置决策：第一类产品（变体结构）。
- DoD：可建带变体+图的商品；图落 media、DB 存相对路径；金额=分、ID=Hashid；磁盘将满停新上传。
- 验收：建带 2 变体+图的商品→刷新仍在→图已压缩/去 EXIF；模拟磁盘紧张验证停上传不影响下单。

### M2 · 店面 + 购物车 + 下单（防超卖）
Storefront API + 内嵌默认主题（字体/图标仅开放许可）；列表/详情；购物车状态机；下单(未付)；**库存预留防超卖**；SEO 基建(JSON-LD/sitemap/meta)。
- DoD：浏览→加购→下单；并发不超卖；页面有结构化数据。
- 验收：前台下单生成订单；并发抢同一库存不超卖。

### M3 · 支付路由 + Stripe/PayPal + 沙箱 + 退款
支付路由层；Stripe/PayPal provider；Webhook 双校验；测试模式；退款；向导支付步骤。
- 前置决策：主攻市场。
- DoD：沙箱跑通一笔支付、双校验通过；退款走通。
- 验收：填测试密钥→沙箱支付成功→订单已付→退款成功。

### M4 · 自动 HTTPS + 向导完整 + 30 分钟开店
内嵌 autocert；向导域名/SMTP(可跳过)步骤；demo 数据；邮件队列(不阻塞下单)。
- DoD：填域名自动签发 HTTPS；向导一气呵成；**北极星跑通**。
- 验收：真实域名走完整向导→HTTPS 店面可访问；全程 ≤ 30 分钟。

### M5 · 数据导入 + 诊断页 + 备份/导出/升级
通用 CSV 导入；Shopify CSV + **301 自动生成** + 图片抓回本地(SSRF 防护)；诊断页(含磁盘/媒体占用)；全量导出 zip；本地自动备份 + **异地备份目标**；升级迁移+回滚。
- DoD：从 Shopify CSV 导入并生成 301；导出可另机恢复(重输主口令)；异地备份连通；升级数据无损。
- 验收：导入 CSV→图本地化/变体保留/301 生成；导出 zip 另机恢复；配异地备份验证；换版本数据无损。

### M6 · v1.1 硬化 + 验收
审计日志、后台 i18n、法律模板、性能边界提示、二进制签名/公证(可后置)、WooCommerce 导入、S3/R2 存储后端、文档。
- DoD：通过 `ARCHITECTURE.md §18` 全部验收项。
- 验收：逐条跑 v1 验收清单。

> **v1 = M0～M5（= 开源版内核全部）；v1.1 = M6。** SaaS 控制面/配额强制/计费/密钥托管/二级域名自动化 = **v2+/商业版，不在本路线内。**

## 8. 风险登记
| 风险 | 影响 | 应对 |
|---|---|---|
| 数据层选型不当 | 换库代价大 | M0 前对比评估 |
| 升级搞坏商家数据 | 信任崩塌 | 自动备份+幂等迁移+回滚，M5 重点验 |
| 备份与数据同盘 | 数据全丢 | 异地备份（M5） |
| 磁盘写满致 SQLite 损坏 | 店瘫/数据坏 | 磁盘监控+优雅降级（M1） |
| 主口令丢失 | 配置不可解 | "重置并重填密钥"兜底 |
| 内核写进 SaaS 专属逻辑 | 双重维护/形态污染 | 接口+默认实现，控制面后置 |
| 范围蔓延 | 交付拖延 | 严格里程碑，超范围回报 |
| 支付回调状态不一致 | 漏单/重复发货 | Webhook 双校验+幂等 |
| 字体/图标许可不清 | 卖企业/开源受阻 | 仅开放许可+LICENSES.md |
| 二进制被杀软误报 | 下载即流失 | 签名/公证（M6） |
